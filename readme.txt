# Secret Santa Project

This project is a Secret Santa application that assigns gift-givers and receivers based on a list of users.

## Dependencies

Ensure you have the following dependencies installed to build and run the project:

- [.NET Core SDK](https://dotnet.microsoft.com/download): .NET Core Software Development Kit
- [CsvHelper](https://www.nuget.org/packages/CsvHelper/): A library for reading and writing CSV files in .NET.
- [Microsoft.Extensions.Configuration](https://www.nuget.org/packages/Microsoft.Extensions.Configuration/): Configuration library for .NET Core applications.
- [Microsoft.Extensions.DependencyInjection](https://www.nuget.org/packages/Microsoft.Extensions.DependencyInjection/): Dependency injection library for .NET Core applications.
- [Microsoft.Extensions.Logging](https://www.nuget.org/packages/Microsoft.Extensions.Logging/): Logging library for .NET Core applications.
- [Microsoft.Extensions.Logging.Console](https://www.nuget.org/packages/Microsoft.Extensions.Logging.Console/): Console logger provider for .NET Core logging.

## Project Structure

The project is structured as follows:

- `SecretSanta`: Main project containing Secret Santa logic and services.
- `SecretSanta.Tests`: Unit tests for the Secret Santa project.
